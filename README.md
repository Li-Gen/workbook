# 使用Node.js实现一个todo功能

## 主要功能

1. 运行`node cli` : 可以查看当前所有的todo

2. 主要子命令: 

    - `node cli add + todo名字` : 可单个添加,也支持一次添加多个todo
    
    - `node clear` : 一次清除所有的todo
    
3. 提供的一级子选项:
    
    - 通过上下键选择todo的项: 展示二级子选项
    
    - 添加新任务: 输入新标题即可添加新的todo
    
    - 退出: 退出命令行
    
4. 二级子选项:

    - 退出, 退出命令行
  
    - 已完成: 将当前选择的todo项标记为已完成
  
    - 未完成: 将当前选择的todo项标记为未完成
  
    - 删除: 将单签选择的todo删除
  
    - 修改标题: 修改当前选择的todo的标题
